# R6 Tracker 翻译器

此插件只作用于 `https://r6.tracker.network/r6siege/profile/ubi/*`。

---

## 它能做什么？

### 1）精确翻译可见页面
`window.exactDict` 维护了一个“英文 → 中文”的对照表。页面上出现这些词的时候，会自动替换成所给的中文

比如：
- Overview → 概览  
- Matches → 比赛  
- Win Rate → 胜率  
- ……

---

## 特殊在哪？

### 1) 支持 # 模糊匹配
支持用 # 作为数字可变对象作为模糊匹配

例子：
- "Round #": "第 # 回合"

### 2) 我被坦克轧过，你被轧过吗？
支持完美小字适配（仅需调整 dict 顺序）

---

## 怎么改词典？

你只需要维护：

### 翻译词典：`window.exactDict`(in dict.js)
```js
window.exactDict = {
  "Overview": "概览",
  "Matches": "比赛",
};
```
---

## Tips

- `window.exactDict` 里如果同一个 key 写了两次，越靠后，优先级越高。
- 类似日期、回合数、`5 W / 2 L` 这类“带变量”的文字，不太适合写成 `"Jan #"` 这种形式；要做的话需要另外加规则，不然匹配不到。
- 实际上是因为在 F12 中可以看到，是切成了两个部分，一个是 5，一个是 W，实际上不可以匹配
